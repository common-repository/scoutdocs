== Description ==

ScoutDocs was a file sharing plugin that runs on your site, but stores your files securely in the cloud.

== Installation ==

ScoutDocs is shutting down. Please don't install it.

== Frequently Asked Questions ==

= I have questions about the shutdown of ScoutDocs =

Send an e-mail to info@scoutdocs.com

== Screenshots ==

1. The file list. Here you can see files that you’ve uploaded or that have been assigned to you by someone else.
2. The drop zone. Either click the “+ Add” button or drag files onto ScoutDocs to get a file drop zone. Drop files here to upload them securely to the cloud.
3. File upload progress.
4. Manage viewers. Here you can add people (or groups of people) to a file. They will be able to see the file and will receive an e-mail notification that you have shared the file with them. You can also choose whether to ask users to accept or decline the file.
5. The ScoutDocs settings screen. You can enter your API key here and change settings about how the ScoutDocs plugin works.
